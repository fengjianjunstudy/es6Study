Module

设计思想：尽量静态化，使得编译时就能确定模块的依赖关系，以及输入和输出的变量

CommonJS 和 AMD都是运行时确定

输入：import 输出：export

import/export可以出现在模块的任何位置，只要在模块的顶层即可，不能处于块级作用域

一个模块就是一个独立的文件

as 关键字可以起到重命名的作用，在import 或 export的时候

import具有提升的作用，会提升到整个模块的头部首先执行

整体加载:

import * as ... from modulePath

module ... from modulePath

export default 指定默认输出，在其他模块加载该模块时，import命令可以为该匿名函数指定任意名字
